Sizing

What size should I buy
HI

We strongly recommend using the sizing chart in the description of each product page to find the size that you like. Our apparel is standard fit—not slim or oversized. It helps to measure out a top or jumper that you currently own to use alongside the sizing chart. Please note that all sales are final—we are not able to offer returns/exchanges due to wrong sizes.

What if I bought the wrong size?
HI

Please note all sales are final—we are not able to offer returns/exchanges due to wrong size or general personal preferences. This is why we strongly recommend using the sizing chart in the description of each product page!
